# Sample Inputs

This folder is intended to contain sample firmware files or documented references for analysis.

> Do not include proprietary or legally restricted firmware images here unless you have explicit rights to share them.

Suggested sample files:

- `dji_firmware_example.bin` — placeholder for a DJI firmware image
- `archive_example.zip` — sample container format for extraction testing

A sample generator is available at `samples/generate_samples.py` to recreate the supported firmware stubs and populate `samples/input` for end-to-end engine testing.
